# Random Password Generator

Приложение на Python с графическим интерфейсом (Tkinter) для создания безопасных паролей.

## Автор
**Юрий Кадзоков**

## Функционал
- Настройка длины пароля (от 4 до 32 символов) с помощью ползунка.
- Выбор состава пароля: цифры, буквы, спецсимволы.
- Валидация: приложение не даст создать пустой пароль без выбранных типов символов.
- История: все сгенерированные пароли сохраняются в `password_history.json` и отображаются в таблице.

## Инструкция по запуску
1. Склонируйте репозиторий.
2. Убедитесь, что Python установлен.
3. Запустите файл командой:
   ```bash
   python main.py
   ```

## Тестирование (Примеры)
1. **Позитивный тест**: Выбрать длину 16, включить все чекбоксы -> Нажать "Генерировать" -> Получить сложный пароль в поле и в таблице.
2. **Негативный тест**: Выключить все три чекбокса -> Нажать "Генерировать" -> Получить предупреждение "Выберите хотя бы один тип символов".
3. **Проверка сохранения**: Сгенерировать пароль, закрыть программу, открыть снова -> Пароль должен остаться в таблице истории.
